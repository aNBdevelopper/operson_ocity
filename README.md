#项目名称：一人一城
* 项目描述

     这是一个房源发布项目。客户可以搜索自己想要的房源或者发布自己的房源。(版本一)
# 项目使用
* 下载
  - `git clone http://github.com/aNBdevelopper/operson_ocity.git`

* 进入
  - `cd operson_ocity`

* 安装依赖
  - `npm install`

* 开发环境
  - 启动
    - `gulp`

  - 访问
    - `http://127.0.0.1:9800`

* 部署环境
  - 构建
    - `gulp build`

  - 启动
    - `gulp

  - 访问
    - `http://127.0.0.1:16088`
